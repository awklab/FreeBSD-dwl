# FreeBSD-dwl

Configuration files and scripts for running **dwl** 0.8 on **FreeBSD** 15.0.  

Includes setup for **slstatus**, **foot**, **Neovim**, and **mako**, as minimal Wayland components.

For a detailed walkthrough, installation instructions, and screenshots, see my blog post: 

[FreeBSD and dwl on a 2010 ThinkPad](https://awklab.com/freebsd-dwl)
